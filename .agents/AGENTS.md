# Piranha Project Agent Rules

## UI/UX Pro Max Standard
All frontend React and Tailwind CSS components MUST adopt the `ui-ux-pro-max` skill standard:
- Implement rich, modern aesthetics (dark themes, glassmorphism, curated color tokens).
- Add micro-animations and distinct hover/active/disabled states to all interactive elements.
- Never mark a feature "Done" until automated Playwright E2E tests execute and pass green.
